# Grace Community Church - Demo Website

## Overview
This is a high-quality demo website for Grace Community Church, created by KaynAero Technocrats to demonstrate best practices for church websites in Kenya. The website addresses common weaknesses found in poorly designed church websites while focusing on creating an excellent first impression for visitors.

## Key Features

### 1. Strong Visual Hierarchy
- Clear, prominent headings that stand out
- Logical content flow from most important to detailed information
- Consistent typography and spacing

### 2. Specific, Non-Generic Copy
- Avoids vague religious clichés
- Provides concrete information about the church
- Uses welcoming, accessible language

### 3. Clear Calls to Action
- Primary CTA: "Plan Your Visit"
- Secondary CTAs throughout the site
- WhatsApp integration for immediate connection
- Directional CTAs on every major section

### 4. Trust Signals
- Professional design aesthetic
- Clear service times and location
- Leadership team section
- Upcoming events calendar

### 5. Mobile-First Responsive Design
- Fully responsive on all devices
- Mobile-friendly navigation
- Optimized touch targets

### 6. Accessibility Features
- Semantic HTML structure
- ARIA labels where appropriate
- Sufficient color contrast
- Keyboard navigable
